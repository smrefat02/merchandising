# Merchandising Package

Sales Contract Management System for Laravel.

## Installation

The package is auto-discovered by Laravel. Run migrations after installation:

```bash
php artisan migrate
```

## API Endpoints

### Buyers

- GET /api/merchandising/buyers
- POST /api/merchandising/buyers
- GET /api/merchandising/buyers/{id}
- PUT /api/merchandising/buyers/{id}
- DELETE /api/merchandising/buyers/{id}

### Sales Contracts

- GET /api/merchandising/sales-contracts
- POST /api/merchandising/sales-contracts
- GET /api/merchandising/sales-contracts/{id}
- PUT /api/merchandising/sales-contracts/{id}
- DELETE /api/merchandising/sales-contracts/{id}
